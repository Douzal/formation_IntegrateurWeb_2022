Les balises containers sémantiques

  • header : l'en-tête
  • main : le contenu principal
  • footer : le pied de page
  • section : une grande partie de page
  • aside : du contenu dont l'importance est secondaire
  • article : un contenu indépendant
  • nav : regroupe les éléments de navigation (liens internes)
